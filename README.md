# ✈️ Flight Booking System

A Java-based Flight Booking System that enables users to view available flights, reserve seats, and manage flight bookings through a simple and interactive console-based interface.

Developed as part of the **Pinnacle Labs Java Development Internship**.

---

## 📌 Project Overview

The Flight Booking System is designed to simulate the core functionalities of an airline reservation platform. Users can browse available flights, book seats, and manage reservations while the system tracks seat availability and booking information.

This project demonstrates Object-Oriented Programming (OOP), data management, user interaction, and reservation system design using Java.

---

## ✨ Features

- ✈️ View available flights
- 🎫 Book flight tickets
- 💺 Seat reservation management
- 📊 Real-time seat availability tracking
- 💰 Ticket price display
- 🧾 Booking confirmation
- 🖥️ Menu-driven console interface
- 🏗️ Object-Oriented Programming (OOP) architecture

## ⚙️ How It Works

1. The system displays all available flights.
2. Users select a flight using its Flight ID.
3. The system checks seat availability.
4. If seats are available, a booking is confirmed.
5. The seat count is automatically updated.
6. Users can continue booking or exit the system.
